Detailed Description

High level Movella DOT management class.

Most functions return a boolean indicating success, a null-object, empty lists or nothing at all. In some cases more in-depth knowledge of the error is required (

See also
loadInputFile). Therefore they return an XsResultValue
Note
This object cannot be copied. The copy constructor has been disabled.

bool XsDotConnectionManager::setPreferredBluetoothAdapter ( XsString  adapterName)
inline

[LINUX ONLY] Set the preferred Bluetooth Adapter used for interaction with devices using the adapter name. Set the preferred adapter BEFORE enabling device detection.

For example, when you have hci0 and hci1 you can set your preferred to hci1.

Returns
True if the preferred adapter has been set. False if failed or not available.
Note
Due to Operating System functionality this function is not available on Windows.
