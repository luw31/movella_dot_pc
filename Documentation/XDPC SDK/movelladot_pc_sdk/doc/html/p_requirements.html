Requirements for Movella DOT Device API

Platform and Other Requirements

Theoretically the API could work on multiple versions of Linux and Windows. However, if you want to be sure that the Movella DOT Device API will be built and run properly, your system should meet the following minimum requirements:

  • Operating system: Ubuntu 20.04 LTS or 22.04 LTS 64-bit Linux, Windows 10 (Build 1703 or higher) or Windows 11.
  • Bluetooth Adapter with Bluetooth 4.0 or higher

Recommended Platform and Adapter Requirements

The requirements listed below are the recommended requirements for an optimal experience using the Movella DOT PC SDK.

  • Operating System: Ubuntu 22.04 64-bit or Windows 10 (Build 1703 or higher)
  • Bluetooth Adapter with Bluetooth 5.0 or higher (Laird BT851)

Supported Interface Requirements

  • C++ Version: C++11
    • C/C++ Compiler: GCC 5.4.0 or MSVC 14
  • Python version 3.7 or higher
  • Java 8 or higher (1.8.0+)
  • C# 7.3

To open the example solutions (Python and Java excluded) use Microsoft Visual Studio 2019 or later.
